# 📝 To-Do List Project (Python)

## 📌 Overview
A simple **To-Do List program** written in Python.  
You can add, view, and delete tasks. All tasks are saved in a file (`tasks.txt`) so they remain even after you close the program.  

## ▶️ How to Run
1. Install Python 3  
2. Run the project:
   ```bash
   python main.py
